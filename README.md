# Hallucination Detector  

Детектор галлюцинаций для больших языковых моделей, использующий внутренние сигналы (логиты, энтропию, скрытые состояния слоёв, Logit Lens). Проект воспроизводит бейзлайн, добавляет новые признаки и проверяет их на датасете с мутациями фактов.

It-purple hack, задача от Сбера


## Описание 

1. **Извлечение признаков** (`src/features/hidden_extractor.py`):  
   Через forward-pass модели собираются:
   - `uncertainty` (19 признаков): статистики logprobs, энтропия, тренд, разрыв top1-top2, коэффициент вариации.
   - `internal_scalars`: нормы скрытых состояний и энтропия Logit Lens на выбранных слоях.
   - `probe_vec`: вектор последнего слоя в момент генерации ответа.

2. **Классификация** (`src/models/classifier.py`):  
   Логистическая регрессия с `SelectKBest` (mutual information) и `StandardScaler`. Работает поверх объединённых признаков.

3. **Инференс** (`src/predict.py`):  
   Класс `GuardianOfTruth` оборачивает пайплайн: текст → forward → признаки → предсказание. Overhead < 30 мс (p99).

**Добавленные признаки**

1) Коэффициент вариации (cv)-std/mean log prob (показывает насколько неравномерна уверенность)
2) Тренд энтропии (slope)-наклон энтропии по токенам(галлюцинации часто имеют растущую энтропию)
3) Количество пиков-число токенов с энтропией >90%(резкие скачки неуверенности)
4) Энтропия первого/последнего токена и их разница(сравнение уверенности в начале и конце)
5) STD Logit Lens энтропи-разброс энтропии по позициям(+мера нестабильности)
6) Средняя max вероятность Logit Lens-насколько модель уверена в лучшем варианте (+сигнал уверенности)
7) STD норм скрытых состояний-разброс норм во время генерации-(аномалии в динамике)


## Результаты
Оценка на публичном бенчмарке (`knowledge_bench_public.csv`). Обучение проводилось на кастомном датасете (~3000 пар).

| Модель / Конфигурация          | Train PR-AUC | Test PR-AUC | Forward (мс) | Overhead p99 (мс) |
|--------------------------------|--------------|-------------|--------------|-------------------|
| Qwen `Qwen3-1.7B` + бейзлайн пайплайн | 0.6475       | 0.5297      | 46.9         | 99.3              |
| Qwen `Qwen3-1.7B` + улучшенный пайплайн | 0.6396       | 0.5412      | —            | —                 |
| **GigaChat3-10B-A1.8B-bf16** + улучшенный пайплайн | **0.7135**   | **0.6255**  | **173.5**    | **30.0**          |

все конфигурации проходят лимит `overhead p99 < 100 ms`.

## Датасет и подготовка данных

- **База**: `ruFacts` (HuggingFace). Отфильтрованы примеры с меткой `is_hallucination=0` (только фактологически верные ответы).
- **Мутации**: Класс `FactualMutator` (`src/data/dataset_builder.py`) создаёт контролируемые искажения: замена сущностей (имена, география), чисел, единиц измерения, отрицаний, квантификаторов и дат.
- **Парная структура**: Каждый промпт хранит две строки: оригинальный ответ (`0`) и мутированный ответ (`1`). 
- **Итог**: ~3000 примеров, сбалансированных по классам


## Быстрый старт
Все пути прописаны относительно корня проекта. Запускать из корневой директории.

```bash
# 1. Установка зависимостей
pip install -e .

# 2. Скачивание модели (по умолчанию GigaChat3-10B-A1.8B-bf16)
./scripts/download_model.sh

# 3. Подготовка датасета (загрузка ruFacts + генерация мутаций)
./scripts/prepare_data.sh

# 4. Обучение классификатора
./scripts/train.sh

# 5. Инференс на публичном бенчмарке
./scripts/private_score.sh
```
Результаты на тесте сохраняются в:
1) data/bench/preds.csv с колонками pred_proba, t_model_ms, t_overhead_ms
2) data/bench/knowledge_bench_private_scores.csv (только pred_proba)
