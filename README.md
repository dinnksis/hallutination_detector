# Hallucination Detector

Детектор галлюцинаций для больших языковых моделей, использующий внутренние сигналы (логиты, энтропию, скрытые состояния слоёв, Logit Lens). Проект воспроизводит бейзлайн, добавляет новые признаки и проверяет их на датасете с мутациями фактов.

**It-purple hack, задача от Сбера**

## Описание

- Извлечение признаков (`src/features/hidden_extractor.py`):
  Через forward-pass модели с помощью хуков собираются:
  - `uncertainty` (19 признаков): статистики logprobs, энтропия, тренд, разрыв top1-top2, коэффициент вариации
  - `internal_scalars`: нормы скрытых состояний (mean + std) и энтропия Logit Lens (mean + std) на выбранных слоях
  - `probe_vec`: вектор последнего слоя в момент генерации ответа

- Классификация (`src/models/classifier.py`):
  Логистическая регрессия с SelectKBest (mutual information) и StandardScaler

- Инференс (`src/predict.py`):
  Класс `GuardianOfTruth` оборачивает пайплайн: текст → forward → признаки → предсказание. 

## Добавленные признаки (поверх бейзлайна Сбера)

| Признак | Описание |
|---------|----------|
| Коэффициент вариации (CV) | std/mean log prob — неравномерность уверенности |
| Тренд энтропии (slope) | наклон энтропии по токенам (галлюцинации → растущая энтропия) |
| Количество пиков | число токенов с энтропией >90 перцентиля |
| Энтропия первого/последнего токена + разница | сравнение уверенности в начале и конце |
| STD Logit Lens энтропии | разброс энтропии по позициям (нестабильность) |
| Средняя max вероятность Logit Lens | уверенность модели в лучшем варианте |
| STD норм скрытых состояний | разброс норм во время генерации (аномалии в динамике) |

## Датасет и подготовка данных

- **База:** RuFacts (HuggingFace). Отфильтрованы только фактологически верные ответы (label=0)
- **Мутации:** класс `FactualMutator` создаёт контролируемые искажения:
  - отрицания (39.7%), единицы измерения (16.8%), квантификаторы (11.4%)
  - география (10.8%), числа (9.9%), годы (5.1%), имена (2.8%), даты (1.9%), дроби (1.7%)
- **Парная структура:** каждый prompt → оригинал (0) + мутация (1)
- **Итог:** 1778 примеров, баланс 50/50
